# EmployeeOS v1 Architecture

EmployeeOS v1 is a local-first company brain with two user surfaces:

- Terminal app for operators
- Desktop app for daily and executive review

Shared runtime packages:

- AI provider abstraction
- Event bus
- Memory model
- Database layer
- Observer, planner, executor, learner, reporter services
- MCP plugin registry

Core data shape:

- Observation
- Decision
- Action
- Outcome
- Learning
- Pattern
